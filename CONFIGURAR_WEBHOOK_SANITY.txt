CONFIGURAR WEBHOOK SANITY -> SUPABASE
====================================

Objetivo:
Que cada producto nuevo o editado en Sanity se sincronice automaticamente a Supabase para poder comprarlo.

URL DEL WEBHOOK:
https://aadjazenpqldtvnxqksr.supabase.co/functions/v1/sync-products

METODO:
POST

HEADER OBLIGATORIO:
Authorization: Bearer TU_VITE_SUPABASE_ANON_KEY

TIPO DE CONTENIDO:
Content-Type: application/json

FILTRO RECOMENDADO EN SANITY:
- Solo documentos de tipo product

PASOS EN SANITY:
1. Entra en manage.sanity.io
2. Abre el proyecto de Jándula Moda
3. Ve a API
4. Busca Webhooks
5. Crea un webhook nuevo
6. Pon la URL indicada arriba
7. Añade el header Authorization con la anon key
8. Configura eventos de create, update y delete para product
9. Guarda

RESULTADO ESPERADO:
- Producto nuevo en Sanity -> se crea/actualiza en Supabase
- Producto editado en Sanity -> se actualiza en Supabase
- Producto borrado en Sanity -> se desactiva en Supabase en la siguiente sincronizacion

NOTA:
Sin este webhook, un producto nuevo puede verse en la web pero no estar listo para comprarse si no existe todavia en la tabla products de Supabase.
